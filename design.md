# DevGenius AI – Design

## Architecture (3-Tier)

Frontend → Backend → Database
               ↓
            AI Engine

---

## Main Modules

1. Authentication Module
2. Adaptive Learning Engine
3. AI Debugger
4. Practice Generator
5. Growth Score Calculator
6. Dashboard & Analytics

---

## Working Flow

1. User logs in
2. Writes code
3. Backend sends code to AI
4. AI analyzes errors & weak areas
5. Practice generated
6. Growth score updated
7. Dashboard shows progress

---

## Database Collections

Users
- user_id
- skill_level
- goals

Submissions
- code
- errors
- topic

GrowthMetrics
